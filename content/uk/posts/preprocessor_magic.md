Як випливає з назви, препроцесори - це програми, які обробляють наш вихідний код перед компіляцією. Існує ряд кроків між написанням програми та виконанням програми в `C`/`C++`. Давайте подивимось на ці кроки, перш ніж ми фактично почнемо дізнаватися про прецесорів.

```flowchart
write=>start: C++ file
programm=>end: Виконуваний файл
preproces=>operation: Препроцесинг
compile=>operation: Компіляція
directive=>condition: Присутні
директиви?
link=>operation: Лінковка

write->directive(no,down)->compile->link->programm
directive(yes,buttom)->preproces(right)->compile
```

Програми препроцесорів надають директиви препроцесорів, які говорять компілятору попередньо обробити вихідний код перед компіляцією. Усі ці директиви препроцесорів починаються з символу "#" (хеш). Символ "#" вказує на те, що будь -яке твердження починається з "#", перейде до програми препроцесора, щоб виконати. Прикладами деяких директив препроцесорів є: #include, #define, #ifndef і т. Д. Пам'ятайте, що символ #забезпечує лише шлях до препроцесора, а команда, як включена, обробляється програмою препроцесорів. Наприклад, #include включатиме додатковий код у вашій програмі. Ми можемо розмістити ці директиви препроцесорів у будь -якій точці нашої програми. -->


Існує 4 основні типи директив препроцесорів:

Макрос
Включення файлів
Умовна компіляція
Інші директиви

Давайте тепер детально дізнаємось про кожен із цих директив.

### define
Макроси - це фрагменти коду в програмі, якій надається деяке ім’я.Щоразу, коли це ім'я зустрічається компілятором, компілятор замінює ім'я фактичним фрагментом коду. Директива `#define` використовується для визначення макросу. Давайте тепер зрозуміємо макрос -визначення за допомогою програми:

```cpp {linenos=false}
#include <iostream>
#include <stdlib.h>
 
#define LIMIT 5
int main()
{
    for (int i{0}; i < LIMIT; i++) {
        std::cout << i << "\n";
    }
 
    return EXIT_SUCCESS;
}
```

Після препроцесингу компілятору буде передано приблизно такий код
```cpp {linenos=false}
#include <iostream>
#include <stdlib.h>
 
int main()
{
    for (int i{0}; i < 5; i++) {
        std::cout << i << "\n";
    }
 
    return EXIT_SUCCESS;
}
```
