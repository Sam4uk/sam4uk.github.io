# Mustek 1248 UB

## Придбання

Ось такий сканер мені дістався в подарунок від сусідів. Хто були моїми сусідами. Нашими сусідами, коли я працювава комапнії `Octopus Energy`, була `NoNameCompany`, яка займалась виготовленням металопластикових вікон. Цю компанію попросили залишити приміщення оскільки вони не могли сплачувати за оренду. Щоб їм було легше переїздити мені подарували даний сканер.

## Підготовка до роботи

Для підготовки для роботи сканера нам потрібно виконати кілька найпростіших дій у терміналі

Переходимо за таким шляхом `/usr/share/sane/gt68xx`
```zsh {linenos=false}
cd /usr/share/sane/gt68xx
```
Якщо отримуєте повідомлення що каталогу не існує то створюємо його. Але для цього треба володіти правами адміністратора системи, адже `sudo` запитає пароль суперкористувача. Якщо пароль вам невідомий - продовжувати нема сенсу. 
```zsh {linenos=false}
sudo mkdir -p  /usr/share/sane/gt68xx
```
Після створення каталогу повторюємо перехід. Після переходу вводимо команду завантаження файлу
